\section*{Run}

\begin{lstlisting}

\end{lstlisting}

Open in browser: \texttt{http://localhost:8080}

\section*{Stop}

In the same terminal where the server is running:

\begin{lstlisting}
Ctrl + C
\end{lstlisting}

If port 8080 is stuck:

\begin{lstlisting}
netstat -ano | findstr :8080
taskkill /PID <PID> /F
\end{lstlisting}
